This is a plugin that collapse a HTML box by using the CSS 3 transitions for the accordion opening/closing animation

This plugin, obviously, does not show the opening/closing animation for the hidden boxes on older browsers like Internet Explorer 7/8/9 because they does not support the CSS 3 transition rules.

The behavior on these browsers will be a simple hide/show of the hidden content.

Remember that you have to adjust in the css the max-height of your boxes to be sure that your hidden content will be visible for any resolution (you can put a very high value, but remember that the closing animation starts from this value when it collapses the box, so maybe it could be very slow).
